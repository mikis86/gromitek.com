# Images needed

Drop your own photos into this folder using these exact filenames (referenced from the CSS/HTML). Landscape orientation, roughly 1600×900px or larger, works best for all of them.

| Filename | Used on | Suggested content |
|---|---|---|
| `hero-plant.jpg` | Home — hero background | Wide shot of the plant / production line |
| `pulpit-bg.jpg` | About — feature block | Pulpit / control room |
| `electrical-cabin.jpg` | About — feature block | Electrical cabinet / cabin |
| `steel-bundles.jpg` | Services — feature block | Finished steel bundles |
| `steel-coils.jpg` | Services — feature block | Finished steel coils |

If a file is missing, the site still looks complete — each block falls back to a solid navy/azure gradient, no broken-image icons.

If any of these were taken on a client site, double-check they don't show confidential layouts, equipment tags, or anything covered by an NDA before publishing.
